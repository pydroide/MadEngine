# MadEngine 📖✨

MadEngine is an entertaining terminal game built in Python that generates hilarious, randomized short stories based on user inputs.

## Features
- **Interactive Prompts:** Choose descriptive traits, characters, locations, and actions to shape the narrative.
- **Randomized Outcomes:** Selects from multiple story templates for varied comedic results.
- **Persistent History:** Saves all generated stories to both memory and a text file (`madEngineHistory.txt`).
- **Game Controls:** Built-in options to replay, view history, clear logs, or exit cleanly.

## How to Run
`python madEngine.py`
